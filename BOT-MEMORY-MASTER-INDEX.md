## Shared Policy

Each bot must store durable teaching, reusable project rules, long-term preferences, corrections, decisions, and lessons learned inside its own folder.

Use the Telegram bot username as the folder name. Do not use internal profile names such as gateway, codexbot, dawny, or nan numbers as memory folder names.

## Graph Notes

Memory is intentionally split into atomic topic notes so Obsidian graph view shows meaningful clusters instead of one large import node.

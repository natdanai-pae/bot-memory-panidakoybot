# Panida Memory Rules

- Always save to both Honcho memory and Obsidian vault when the user teaches something new, gives preferences/habits, corrects the assistant, or says “remember this”.
- Save user preferences, environment facts, and conventions to memory.
- Save detailed procedures, workflows, research, and documentation to Obsidian.
- Use wikilinks to connect notes.
- Offer to save complex workflows as skills.
